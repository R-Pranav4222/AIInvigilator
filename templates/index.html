Features

Powerful Detection Suite

Seven specialized detection algorithms work in concert to identify every form of exam malpractice.

Mobile Detection

Identifies smartphones and electronic devices using YOLO object detection with smart proximity filtering.

Turning Back

Detects students looking at neighboring desks using head pose estimation and shoulder alignment analysis.

Paper Passing

Multi-person proximity analysis with hand-to-hand trajectory tracking for cheat sheet exchanges.

Leaning

Detects excessive leaning toward neighbors using torso angle analysis and MediaPipe pose landmarks.

Hand Raise

Identifies suspicious hand-raise gestures that may be used for signaling answers between students.

AI Probability

Five-factor weighted scoring algorithm analyzes detection confidence, frequency, and behavior patterns.
